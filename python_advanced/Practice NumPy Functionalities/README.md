## Practice on NumPy and Pandas

This practice module reinforces key concepts from previous NumPy and Pandas notebooks. It helps you strengthen your understanding of how to create and manipulate NumPy arrays, perform mathematical operations, extract specific elements through indexing, and apply aggregation methods.

### What You Will Practice
- Creating NumPy arrays  
- Manipulating arrays with basic mathematical operators  
- Extracting rows, columns, and individual items using indexing  
- Using aggregation methods such as sum, min, max, and standard deviation  

---

## Creating NumPy Arrays

You will practice creating arrays in different ways:
1. Create a NumPy array from a Python list.  
2. Create a NumPy array containing values from 1 to 15.  
3. Create a NumPy array from 0 to 8 with a step size of 0.3.  
4. Create a NumPy array with shape (3, 4) containing random values.

---

## Manipulating NumPy Arrays

You will work with array operations and transformations:
1. Create a NumPy array from mixed values `(1, 3, '5', 7, 9, '10')` and cast everything to integers.  
2. Check the array’s data type and shape.  
3. Update the fourth value to `8`.  
4. Reshape the array into a 2×3 matrix.  
5. Add `8` to each value in the first row and `9` to each value in the second row.  
6. Multiply the first column by `3`, the second by `4`, and the third by `5`.  
7. Compute the row-wise sums for both rows.  
8. Find the smallest value in each column.  
9. Extract the element in the second column of the first row.  
10. Determine the index at which the value `72` occurs.
