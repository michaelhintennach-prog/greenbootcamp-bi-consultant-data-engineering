# Practice Pandas Functionalities

This practice session reinforces the concepts introduced in the Pandas introduction module.  
Using the *white wine quality dataset* (`winequality-white.csv`), you explore, inspect, filter, transform, and analyze data using core Pandas functionality.  
The exercises follow a typical **Exploratory Data Analysis (EDA)** workflow.

## Objectives
- Load and inspect tabular data with Pandas  
- Understand DataFrame structure, data types, and summary statistics  
- Practice selecting, slicing, and filtering data  
- Apply grouping, aggregation, and sorting  
- Create new columns and derive insights  
- Perform basic exploratory visualizations

## Topics Covered

### 1. DataFrame Basics
You practice the standard workflow:
- Importing Pandas  
- Reading CSV data (`sep=";"`)  
- Inspecting structure: rows, columns, data types  
- Checking missing values  
- Generating descriptive statistics (min, mean, max, median)

### 2. Selecting and Slicing Data
Exercises include:
- Selecting specific rows and columns  
- Using `.head()`, `.tail()`, `.loc[]`, `.iloc[]`  
- Filtering with boolean conditions  
- Combining multiple conditions  
- Comparing values to column means

### 3. Grouping, Aggregation & Sorting
You apply:
- `groupby()` to compute averages per category  
- Conditional grouping (e.g., alcohol by pH ranges)  
- Finding maximum/minimum values  
- Sorting to retrieve top/bottom records  
- Creating new derived columns (e.g., `total_acidity`)

### 4. Exploratory Analysis & Visualization
You explore:
- Percentage of high‑quality wines  
- Distribution of alcohol content  
- Counts of wines per quality level  

These tasks combine filtering, aggregation, and basic plotting to gain insights into the dataset.

## Summary
This practice module strengthens your ability to:
- Navigate and manipulate DataFrames  
- Extract meaningful subsets of data  
- Compute aggregated metrics  
- Engineer new features  
- Perform initial exploratory analysis  

It serves as a hands‑on bridge between basic Pandas concepts and more advanced EDA workflows.
