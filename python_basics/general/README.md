# General

This folder contains introductory materials that support the Python Basics module.  
It covers essential tools and practices used throughout the training program.

## Contents
- **Bash Basics** – Working with the command line, navigation, file handling, and processes.  
- **Best Coding Practices** – Writing clean, readable, and maintainable Python code.  
- **Intro to Jupyter Lab** – Using notebooks for interactive development and data exploration.

These fundamentals provide the foundation for efficient work in data engineering and analytics.
