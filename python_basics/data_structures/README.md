# Data Structures

This folder contains materials related to Python’s built-in data structures, which are crucial for handling and transforming data.

## Contents
- **Lists** – Dynamic sequences for storing and processing collections of items.  
- **Sets** – Unique-value collections useful for data cleaning and comparisons.  
- **Mutability** – Understanding how data types behave when modified.  
- **Dictionaries** – Key-value structures widely used for JSON-like data.  
- **Comprehensions** – Efficient and expressive ways to transform data.

Mastering these structures is key to writing efficient and clean data engineering code.
