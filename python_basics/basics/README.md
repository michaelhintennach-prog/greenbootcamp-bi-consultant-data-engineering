# Python Basics

This folder contains core Python concepts that form the foundation of programming in the data engineering context.

## Contents
- **Numeric Variable Types** – Integers, floats, conversions, and basic operations.  
- **Strings** – Manipulation, slicing, formatting, and common methods.  
- **If-Statements** – Conditional logic for controlling program flow.  
- **Loops** – Iterating over data structures and automating repetitive tasks.  
- **Plotting** – First steps in visualizing data using Python.

These topics are essential for understanding how to process and analyze data programmatically.
